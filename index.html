QUICK START GUIDE

  1. Enter your Start Date and Starting Balance above
  2. Go to 'Quick Entry' to add your income and expenses
  3. For each transaction, select Type, Category, Amount, and Frequency
  4. View the 'Calendar' to see daily balances
  5. Check the 'Dashboard' for summary charts

💡 TIP: All projections update automatically when you modify your transactions!

💡 How to sync from Excel:
  1. In Excel, ask Copilot to export your Transactions table to JSON
  2. Copy the JSON and save it as a .json file
  3. Click "Choose JSON File" above and select your file
  4. Your data will be imported and all views will update!
